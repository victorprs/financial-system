    Financial System v0.1.0 FinancialSystem.Account View Source

    Data structure to store account properties.

    Each account has a balance that represents the funds that it has in a FinancialSystem.Money struct and its respective FinancialSystem.Currency. Each account also has an owner with its name represented by the owner string. And of course, an account number to uniquely identify an account.
